Installation

Download the latest releax ISO image and check sha512sum for integerity 
$ sha512sum checksum

Setup VirtualBox

  • download VirtualBox
    • $ sudo apt install virtualbox # debian/ubuntu/...based distro

    $ sudo pacman -S virtualbox # archlinux

    $ sudo dnf install virtualbox # fedora/redhat linux

    * if issue with virtualbox to start then try this 'modprobe vboxdrv' or restart once
  • Create a new linux x86_64 bits virtual machine with at least 1024 MiB Memory and 2048 MiB Storage

  • load releax iso in virtual machine and boot it
    and login with username 'root' and passwd 'root'

  • create (if necessary) the required partitions (at least root partition)
    $ cfdisk /dev/sda # to start disk partition tool
    $ mkfs.ext4 /dev/sda1 # to format the selected disk into ext4

  • execute releax-installer to automate the installation process
    $ releax-installer
    ENV variables ROOTS=/mnt and PACKAGES=/packages to configure build if err comes

  • follow the instruction to complete the installation process,
    installer ask you to enter
    • root password ------------------ # Enter 2 Times
    • Default username ------------- # Small letters with character starting
    • users password ---------------- # Enter 2 Times
    • TimeZone ------------------------- # if you have no idea press enter or type Asia/Kolkata
    • Bootloader ----------------------- # if following the instruction ditto then type /dev/sda

  • Edit /etc/fstab, /etc/bootup/init, /etc/bootup/restart, /etc/hostname or any other thing you want

    • i you didn't saw any error then you have done every thing write and reboot with the command that will shown on the screen

Horry ! if every thing is done correctly and god wants , then Remove the releax iso and reboot the virtual machine
